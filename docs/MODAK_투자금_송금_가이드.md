## 1. 투자금 + 지분 구조

### 주주 구성

```
MODAK Group Pty Ltd
├── 김주영: 지분 90% (경영 의사결정권)
└── ATS Total Service Pty Ltd: 지분 10%
    (투자금 $10,000~20,000 AUD 대가로)
```

### 합의 내용 보관 방법 (이메일)

```
보내는 사람: admin@modak.com.au (법인 이메일)
받는 사람: 매제 이메일

제목: MODAK Group Pty Ltd 지분 관련 합의 내용

MODAK Group Pty Ltd 설립과 관련하여
아래 내용으로 합의했음을 확인합니다.

1. 투자 금액
   ATS Total Service Pty Ltd → $10,000~20,000 AUD

2. 지분율
   김주영: 90%
   ATS Total Service Pty Ltd: 10%

3. 투자금 용도
   MODAK 앱 개발 및 초기 운영 비용

4. 기타 합의 사항
   - 경영 의사결정권은 김주영에게 있음
   - 배당은 추후 수익 발생 시 협의

위 내용에 동의하시면 회신 부탁드립니다.
```

> 매제가 "확인" 또는 "동의" 회신하면 합의 증거로 남음
> 이메일 스레드 PDF 저장 → Google Drive 법인 폴더 보관

## 2. 개발비 해외 송금 — Wise Business

### 왜 Wise인가

| 방법 | 수수료 | 속도 |
|------|--------|------|
| **Wise (추천)** | **0.4~1%** | **1~2일** |
| Western Union | 2~4% | 1~3일 |
| 은행 SWIFT | $20~40 고정 | 3~5일 |
| PayPal | 3~5% | 즉시 |

### 송금 순서

```
1. Wise Business 계정 가입
   (wise.com/business — 법인 명의로)
        ↓
2. 법인 은행 계좌 연결
   (CBA / ANZ / NAB)
        ↓
3. 개발자 한국 계좌 정보 받기
   - 은행명
   - 계좌번호
   - 예금주 이름 (영문)
        ↓
4. AUD → KRW 송금
        ↓
5. 송금 내역 PDF 저장
   (회계 처리용)
```

### 송금 시 메모란 기재 예시

```
1차 선금:   "MODAK App Development - 1차 선금"
2차 중도금: "MODAK App Development - 2차 중도금"
3차 잔금:   "MODAK App Development - 3차 잔금"
```

### 비용 예시 (3,000만원 기준)

| 회차 | 금액 | Wise 수수료 (약 0.6%) |
|------|------|---------------------|
| 1차 선금 (30%) | 900만원 | 약 54,000원 |
| 2차 중도금 (40%) | 1,200만원 | 약 72,000원 |
| 3차 잔금 (30%) | 900만원 | 약 54,000원 |
| **합계** | **3,000만원** | **약 18만원** |

> 은행 SWIFT 송금 대비 수수료 10배 이상 저렴

## 3. 회계 처리

### 송금 후 Xero 기록

```
송금 내역 PDF 저장
    ↓
Xero에 비용으로 기록
항목: Software Development / IT Consulting
    ↓
회계사한테 전달
```

### 보관할 서류

| 서류 | 용도 |
|------|------|
| Wise 송금 내역 PDF | 비용 처리 증빙 |
| 개발자와 합의한 내용 (카톡/이메일) | 계약 근거 |
| 각 회차별 송금 메모 | 회계 분류 |
